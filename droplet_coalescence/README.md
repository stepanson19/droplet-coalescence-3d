# Слипание пары капель воды в невесомости

Готовое настольное приложение на **Python + Tkinter + Matplotlib**.

В проекте есть:
- **математическая модель**;
- **графический интерфейс**;
- **вычислительный эксперимент** (parameter sweep);
- экспорт симуляции в **PNG/MP4** и результатов эксперимента в CSV.

## Что моделируется

Рассматриваются две одинаковые капли воды радиуса `R`, находящиеся в невесомости.
Сначала растет жидкий мостик между каплями, затем объединенная капля релаксирует к сферической форме и совершает затухающие осцилляции.

Программа использует **reduced-order модель**:
1. ранняя стадия — закон роста перешейка `r_n(t)`;
2. поздняя стадия — затухающая мода `l = 2` для объединенной капли.

Подробные уравнения встроены в вкладку **«Модель и справка»**.

## Запуск на Mac (M1 / M2 / M3)

### Вариант 0 — web-интерфейс (рекомендуется, если Tk не запускается)

```bash
cd droplet_coalescence
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
streamlit run web_app.py
```

После запуска откройте адрес вида `http://localhost:8501`.
В web-версии доступны:
- изменение параметров симуляции (включая `k_slow`);
- вкладка вычислительного эксперимента;
- экспорт PNG/GIF/MP4 и CSV.

Самый простой web-запуск: дважды кликните `run_web.command`.

### Вариант 1 — самый простой

1. Распакуйте проект.
2. Дважды кликните **`run.command`**.
3. Откроется Terminal, создастся виртуальное окружение, установятся зависимости и запустится приложение.

Если macOS ругается на запуск файла:
- откройте Terminal,
- перейдите в папку проекта,
- выполните:

```bash
chmod +x run.command
./run.command
```

### Вариант 2 — вручную

```bash
cd droplet_coalescence
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
python3 main.py
```

## Как пользоваться

### Вкладка «Симуляция»

Задайте:
- `R` — радиус капли, мм;
- `ρ` — плотность, кг/м³;
- `μ` — динамическая вязкость, мПа·с;
- `σ` — поверхностное натяжение, мН/м;
- `r0/R` — начальный радиус мостика;
- `A0` — начальная амплитуда релаксационной моды;
- `k_slow` — коэффициент замедления слипания (чем больше, тем медленнее стадия 1);
- `total_time_ms` — длительность моделирования.

После нажатия **«Запустить»**:
- слева показывается форма капли/капель;
- справа сверху — график роста перешейка `r_n/R`;
- справа снизу — график амплитуды `A(t)`.

Ползунок позволяет вручную просматривать процесс во времени.
Кнопки **«Сохранить PNG»** и **«Сохранить MP4»** экспортируют текущую симуляцию.

### Вкладка «Вычислительный эксперимент»

Можно варьировать один параметр:
- `R`,
- `μ`,
- `σ`,
- `ρ`.

Программа строит:
- график `t_merge`;
- графики `T₂` и `τ_damp`;
- таблицу результатов;
- текстовый комментарий по наблюдаемым тенденциям.

Результаты можно сохранить в CSV.

## Ограничения модели

Это **не** прямой DNS/VOF/Level-Set расчет полной системы Навье—Стокса.
Это упрощенная, но физически осмысленная учебная модель, которая хорошо подходит для:
- курсовой/лабораторной;
- демонстрации влияния параметров;
- визуализации процесса;
- вычислительного эксперимента.

Если понадобится, на основе этого проекта можно сделать и более тяжелую CFD-версию.
