# 3D-модель слипания капель в невесомости — Design

## Цель

Создать отдельное учебное приложение в папке `droplet_coalescence_3d/`, которое моделирует и визуализирует слипание двух одинаковых капель воды в невесомости в 3D, содержит интерактивный интерфейс и поддерживает вычислительный эксперимент.

## Принятые решения

- Базовая платформа: `Python + Streamlit`.
- Визуализация: интерактивная 3D-поверхность в `Plotly`.
- Физическая часть: reduced-order модель, а не CFD/DNS.
- Исходная 2D-версия не изменяется; 3D-решение живет в отдельной папке.

## Математическая модель

Используется двухстадийное описание:

1. Рост жидкого мостика между двумя каплями одинакового радиуса `R`.
2. Релаксация объединенной капли к сферической форме с затухающей модой `l = 2`.

### Стадия 1

Радиус шейки `r_n(t)` определяется reduced-order законом роста с композитной скоростью:

- вязкостно-капиллярная составляющая;
- инерционно-капиллярная составляющая;
- итоговая скорость берется через гармоническое среднее.

Это сохраняет учебную физическую интерпретацию и позволяет получить разумные времена слияния без полного решения уравнений Навье–Стокса.

### Стадия 2

После достижения порога `r_n / R = merge_ratio` объединенная капля описывается как осесимметричное возмущение сферы:

`r(θ, t) = R_eq [1 + A(t) P_2(cos θ)]`

где `P_2` — полином Лежандра второго порядка, а амплитуда `A(t)` удовлетворяет линейному уравнению затухающей моды:

`A'' + 2βA' + ω_0^2 A = 0`.

### Невесомость

Для условий невесомости принимается `Bo = 0`, поэтому гравитационная деформация отсутствует.

## 3D-геометрия

3D-форма строится как тело вращения вокруг оси слияния:

- до слияния: мягкое объединение двух сфер через implicit field в меридиональном сечении;
- после слияния: вращение осесимметричного профиля объединенной капли;
- в переходной зоне: сглаженное смешение пред- и пост-слияния.

Профиль `ρ(x)` извлекается численно из нулевого уровня implicit field, после чего вращается по азимутальному углу.

## Интерфейс

Приложение содержит две вкладки:

- `Симуляция 3D`: параметры, 3D-визуализация, временные графики и сводка.
- `Вычислительный эксперимент`: sweep по одному параметру, графики и таблица результатов.

## Вычислительный эксперимент

Поддерживается варьирование одного параметра из набора:

- `R`,
- `μ`,
- `σ`,
- `ρ`.

Для каждой точки диапазона рассчитываются:

- `t_merge`,
- `τ_c`,
- `τ_v`,
- `T_2`,
- `τ_damp`,
- `Oh`.
